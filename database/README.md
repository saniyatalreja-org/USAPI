# UniSwap Database

## Database Setup

1. Run `schema.sql` to create all database tables.
2. Run `sample_data.sql` to insert sample records.
3. Start the backend server:

bash
npm start

4. Open the frontend in your browser.

## Database Tables

- Users
- Categories
- Universities
- Listings
- Wishlist
- Transactions
- Project Requests
- Exchange Requests
- Messages
- Reviews

Database: PostgreSQL (Neon DB)
